This is simple Kanban Application which contain the following features:

1. A simple password based login system which does not use any kind of login system.
2. Ability to add any number of lists which can be easily modified and deleted.
3. Ability to add tasks to those lists which also can be easily modified and deleted.
4. Ability to move those tasks within different lists.
5. Ability to show no of task completed and pending plus bar chart for no of tasks created in a day.
6. Ability to visually differentiate between completed, pending and late tasks.

To run the application open terminal in the current working directory and type the following command:

>>  python -m flask run
